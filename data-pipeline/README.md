## ⚠️ 라이선스 주의 (CLAUDE.md / SPEC §7)

- **학교알리미 OpenAPI(schoolinfo.go.kr) = 공공누리 제2유형(출처표시-변경금지).**
  영리 서비스의 가공·2차적 저작물 작성이 금지됨. `fetchSchoolinfo.ts` 산출물
  (`schoolinfo_basic*.json`)은 **git 미추적·프로덕션 미사용**. 항목 구조 참조용으로만 보존.
  → 동일 정보(고교유형·남녀공학·설립유형)를 **data.go.kr 15107734(제1유형)**로 교체할 것.
- data.go.kr **표준데이터**는 대부분 제1유형(출처표시, 상업+변경 자유) — 위치·통학구역이 여기 해당.
- 출처·기준일은 리포트에 항상 표기(이미 SchoolFacts.source/asOf로 강제).
