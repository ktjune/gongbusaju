# data-pipeline — 학교·통학구역 데이터 적재

앱과 분리된 별도 배치 파이프라인. 런타임은 조회만 하며 적재에 관여하지 않는다.

## 갱신 주기
- **3월·9월** 공공데이터포털 표준데이터 갱신 시 재실행

## 데이터 소스 (SPEC §7 참조)

| 용도 | 소스 |
|------|------|
| 주소→배정 초등학교 | 전국초등학교통학구역표준데이터 (data.go.kr/data/15021149) |
| 학구↔학교 연계 | 전국학교학구도연계정보표준데이터 (data.go.kr/data/15021158) |
| 학교 위치(좌표) | 전국초중등학교위치표준데이터 (data.go.kr/data/15021148) |
| 학교 기본정보 | 전국초중등학교기본정보표준데이터 (data.go.kr/data/15107734) |
| 진학현황 | 학교알리미 Open API (schoolinfo.go.kr) — **데이터셋 ID 포털에서 확인 필요** |
